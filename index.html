Keyboard shortcuts

  • Alt + x - Show CNXML source
  • Alt + s - Save local copy
  • Alt + r - Restore local copy
  • Esc - Hide extension menu
  • Ctrl + Shift + z - Undo add element action

Alternative operation

  • LMB + Alt + Image - Open image attribute modal (src & alt)
  • LMB + Alt + element in content - Open extension menu if exist
  • LMB + Ctrl + Tool - Add element as a sibling of current element
