Keyboard shortcuts

  • Alt + x - Show CNXML source
  • Alt + s - Save local copy
  • Alt + r - Restore local copy
  • Esc - Hide extension menu
  • Alt + Space - Toggle Equations Panel
  • Ctrl + Shift + z - Undo add element action
  • Alt + LMB + EqButton - Remove equation button from Equations panel

Alternative Content Operations

  • Alt + LMB + Image - Open image attribute modal (src & alt)
  • Alt + LMB + element in content - Open extension menu if exist
  • Alt + LMB + Tool - Add element as a sibling of current element
