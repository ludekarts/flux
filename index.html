Flux 3

Keyboard shortcuts

Add Elements

  • Alt + LMB + Tool - Add element as a sibling of focused element
  • Alt + Ctrl + LMB + Tool - Add element as a last in the content

More Options

  • Alt + LMB on Image - Open image attributes modal
  • Alt + LMB on Equarion - Open LaTeX input modal
  • Alt + LMB on Reference - Open reference attributes modal
  • Alt + LMB om Content's element - Open extension menu (if exist)
  • Alt + LMB on Equarion Button - Remove equation from Equations Panel
  • Ctrl + Space - Toggle Equations Panel
  • Esc - Hide extension menu / Dismiss modals

Save & View

  • Alt + r - Restore local copy
  • Alt + x - Show CNXML source
  • Ctrl + s - Save local copy
  • Ctrl + z - Undo last action (limited to 15 actions)
  • Shift + LMB on Content's element - Copy element ID into the clipboard
